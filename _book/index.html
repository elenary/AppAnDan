Chapter 1 Чек-лист для работы с данными

  1. Сформулированы теоретические гипотезы
  2. Сформулированы эмпирические гипотезы
  3. Есть понимание, как будут выглядеть / выглядят собранные данные
  4. Есть понимание, что данных достаточно для проверки поставленной гипотезы
  5. Исходя из гипотезы и того, как собраны и какие данные, выбран статистический метод проверки экспериментальной гипотезы
  6. Исходя из статистического метода, есть понимание о статистической гипотезе (хорошо, если есть)
  7. Выбрана среда для анализа данных и пакет для этого метода и конкретная функция
  8. Данные приведены в вид, к которому можно применить выбранный статистический метод
  9. Визуализация самих данных для лучшего их понимания
  10. Проведен расчет выбранного стат метода
  11. Вид полученных результатов понятен для интерпретации, и резульатты согласуются с представлением о том, как они должны выглядеть
  12. Результаты проинтепретированы и стедлан вывод относительно изначальной гипотезы
  13. Визуализированы результаты, так, чтобы их было легче правильно проинтерпретировать
